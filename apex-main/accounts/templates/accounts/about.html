Objective

Passwords are ubiquitous today on any platform, on possibly any website. But to remember so difficult passwords and that too on numerous websites seems daunting and therefore you can devise a project illustrating graphical password strategy.

To provide users a graphical password authentication system where instead of creating a alphanumeric password a user has to select graphical objects in a particular order to keep it as their password.

This will allow the user to set passwords in the form of graphical presentation in a certain pattern and later use that pattern to login to the system.

Our Solution

We propose a graphical password authentication system. Research has shown that graphical objects or images are more easier to remember. Human brain tends to storage visual information for a longer period of time as compared to alphanumeric series.

In our solution, users must select specific icons from a the grid in a sequence. The icons are displayed on the screen in a tab. Users will choose icons from two separate tabs. The values assigned to the selected icons will be combined to create a passcode value, which will be saved in the database along with the username.
